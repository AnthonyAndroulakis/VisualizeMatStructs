# VisualizeMatStructs
visualize.py, graph depth of MatLab structs in python     
     
# Prerequisites:
python3     
python modules:
+ mat2obj (https://github.com/AnthonyAndroulakis/mat2obj)    
+ itertools       
+ matplotlib      
+ numpy     
+ scipy      
     
# How to run:
import visualize     
visualize.depth('filename')
